# Aarogya Assist 🩺

Aarogya Assist is an AI-powered healthcare assistant that provides medical report summarization, image analysis, multilingual support, wellness guidance, and AI-based healthcare chat through a premium and responsive interface.

## Features
- AI Medical Chatbot
- Medical Report Analysis
- Medical Image Analysis
- Multi-language Support
- Premium Responsive UI
- OCR-based PDF Text Extraction

## Technologies Used
- Python
- Streamlit
- OCR
- AI APIs
- HTML/CSS
